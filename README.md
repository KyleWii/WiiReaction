WiiReaction is a simple Wii reaction time game made for the Nintendo Wii. You can test your reaction time by waiting for the GO signal and then pressing (A) as quickly as possible. The game tracks your reaction speed, includes a high score system and has settings like light/dark mode, sound toggles and color changing toggles. It is built as a homebrew app.

You can contribute! Just make a pull request.
